## Installation
Ensure Python 3 is installed, and use pip to install this project.

```bash
pip install uv
uv pip install git+https://github.com/UltraProcessor/CheckersV2.git
```

## Usage
After installing using the command above, running the following command
will start the game.

```bash
my_checkers_game
```
